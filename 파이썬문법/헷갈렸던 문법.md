## 1. `range()`의 방향과 범위

`range()`는 기본적으로 양수 방향(`+1`)으로만 움직이려고 합니다.

- **잘못된 예**: `range(-3)` → `0`에서 시작해 `+1`씩 더해서 `-3`에 갈 수 없으므로 실행 안 됨.
- **역순 루프**: `range(start, stop, step)` 형식을 지켜야 함.
  - `range(0, -3, -1)` → `0, -1, -2` (3번 반복)
  - `range(r, -1, -1)` → `r`이 `-2`라면 `-2, -1, 0` 순서로 진행 (방향 인자 `-1` 필수!)
- **단순 반복**: 횟수가 중요하다면 `range(abs(r))`처럼 절댓값을 활용하는 것이 안전함.

## 2. 가변 객체(List, Set)의 공유 문제

함수에 리스트나 세트를 넘길 때, 파이썬은 **주소값(참조)**을 넘깁니다.

- **증상**: 함수 안에서 `infections.add()`를 하면 함수 밖의 원본 데이터도 같이 변함.
- **해결**: 매 반복문마다 독립적인 시뮬레이션이 필요하다면 새 객체를 생성하거나 복사해야 함.

```python
# 매 루프마다 초기화
for comb in all_combinations:
    current_infections = {start_node}  # 새 세트 생성
    infect(current_infections, nodes, comb)
```

## 3. 리스트의 차원 관리 (`append` vs `extend`)

이미 리스트인 결과물을 다시 리스트에 넣을 때 차원이 결정됩니다.

| 메서드 | 동작 | 차원 변화 | 적합한 상황 |
|--------|------|-----------|-------------|
| `append(list)` | 리스트 통째로 한 칸에 삽입 | 차원 **증가** (이중 → 삼중 배열) | 로봇별 경로를 묶어서 관리하는 시뮬레이션 |
| `extend(list)` | 리스트의 원소만 꺼내서 이어 붙임 | 차원 **유지** | 모든 데이터를 한 줄로 평평하게 나열할 때 |

## 4. `set`과 `dict`의 키(Key) 제약

리스트(`[x, y]`)는 내용이 변할 수 있어서 세트의 원소나 딕셔너리의 키가 될 수 없습니다.

- **에러**: `TypeError: unhashable type: 'list'`
- **해결**: 반드시 **튜플**(`(x, y)`)로 변환하여 사용.

```python
# 저장할 때부터 튜플로 넣는 것이 가장 좋음
all_way.append((r, c))

# 혹은 체크할 때 변환
if tuple(loc) in check_crash_set:
    ...
```

## 5. 딕셔너리 그룹화 꿀팁 (`setdefault`)

좌표별로 로봇이나 시간대를 모을 때 `if-else` 문을 줄여줍니다.

```python
dict.setdefault(key, []).append(value)
```

- **의미**: "키가 없으면 빈 리스트 만들고, 있으면 가져와서 추가해라."
- **응용**: 동일 시간, 동일 좌표에 로봇이 여러 대 있는지 체크할 때 (충돌 감지) 유용함.

## 6. 시간 및 숫자 형변환 주의사항

### 시간 계산
문자열 `"01:00:04.002"`는 숫자가 아니므로 반드시 `split` 후 `int`로 변환하여 최소 단위(ms)로 통일한 뒤 계산할 것.

### `int()`의 동작
- `int(12.22)` → **소수점 버림** (올림 아님!) → `12`
- `int("12.22")` → **에러 발생** → `int(float("12.22"))` 순서로 변환 필요

### 혼합 연산
```python
10 + 12.22  # int + float → 자동으로 float 반환: 22.22
```
에러 없이 자동으로 `float`가 됨.

## 7. 문자열 슬라이싱과 `split`

### 슬라이싱
- `t[:-1]`: 문자열의 맨 마지막 한 자리를 제거.
- 활용 예: 마이크로초를 밀리초로 자를 때 `[:-3]` 등 사용.

### `split()` 주의사항
- 구분자가 문자열 내에 **없어도 에러가 나지 않음** → 원본이 담긴 리스트 `[원본]` 반환.
- **단**, 언패킹 시 개수가 안 맞으면 에러 발생:

```python
# 위험한 코드
a, b = s.split()  # s에 구분자가 없으면 ValueError!

# 안전한 코드
parts = s.split()
if len(parts) == 2:
    a, b = parts
```
