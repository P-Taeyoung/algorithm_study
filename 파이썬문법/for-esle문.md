# 파이썬(Python) `for - else` 문법 완벽 정리

파이썬의 `for - else` 구문은 반복문이 **"중간에 끊기지 않고 끝까지 실행되었는지"**를 판별할 때 사용하는 독특한 문법입니다.

---

## 1. 동작 원리
`else` 블록은 `for` 루프와 한 쌍으로 작동하며, 다음과 같은 조건에서 실행 여부가 결정됩니다.

* **실행됨:** 루프가 반복 가능한 객체의 끝까지 정상적으로 도달했을 때.
* **실행 안 됨:** 루프 도중 `break` 문을 만나서 강제로 종료되었을 때.



---

## 2. 기본 구조

```python
for 요소 in 반복가능객체:
    # 반복할 코드
    if 특정_조건:
        break  # 여기서 나가면 else는 실행되지 않음
else:
    # 루프가 break 없이 '완전하게' 끝났을 때만 실행
    print("모든 요소를 확인했습니다.")
