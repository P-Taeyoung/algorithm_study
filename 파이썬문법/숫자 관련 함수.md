# 파이썬(Python) 숫자 관련 핵심 함수 총정리 가이드

파이썬에서 수치 데이터를 처리하고 수학적 계산을 수행할 때 사용하는 핵심 함수들을 정리한 문서입니다.

---

## 1. 기본 내장 함수 (Built-in Functions)
별도의 `import` 없이 즉시 사용 가능한 가장 중요한 함수들입니다.

### 🔢 수치 연산 및 변환
* **`abs(x)`**: 숫자 `x`의 **절대값**을 반환합니다.
* **`pow(x, y)`**: **거듭제곱**($x^y$)을 계산합니다. (`x ** y`와 동일)
* **`divmod(a, b)`**: **몫과 나머지**를 동시에 반환합니다. 결과는 `(몫, 나머지)` 형태의 튜플입니다.
* **`round(x, n)`**: 숫자를 **반올림**합니다. `n`은 소수점 자릿수이며 생략 시 정수로 반올림합니다.

### 📊 집계 및 비교
* **`max(iterable)`** / **`max(a, b, ...)`**: 인자들 중 **최댓값**을 반환합니다.
* **`min(iterable)`** / **`min(a, b, ...)`**: 인자들 중 **최솟값**을 반환합니다.
* **`sum(iterable)`**: 리스트나 튜플 내 **모든 수의 합계**를 구합니다.
* **`len(iterable)`**: 데이터의 **개수(길이)**를 확인합니다.

```python
# 활용 예시
print(abs(-5))          # 5
print(pow(2, 3))        # 8
print(divmod(10, 3))    # (3, 1)
print(max(1, 5, 2))     # 5
print(sum([1, 2, 3]))   # 6
